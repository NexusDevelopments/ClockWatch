# ClockWatch

# Main Website: https://clockwatch.vercel.app/

ClockWatch is a small React stopwatch + timer app (Stopwatch + Timer) with simple animations.

Run locally:

```bash
npm install
npm run dev
```

Open http://localhost:5173

Deploying the site (options):

- GitHub Pages (recommended when you push this repo to GitHub): add and push the repository, then enable Pages or let the included GitHub Action publish a build. The repository must be public or have Pages enabled. The workflow file `.github/workflows/deploy.yml` is included and will publish the built site using GitHub Pages when you push to the repository.
- Vercel / Netlify: Connect the repository and set the build command to `npm run build` and publish the `dist` directory.
